# SPARQL-Utility-Showcases
Various SPARQL Queries that demonstrate practical usefulness to real-world data challenges. Examples include:
* Conceptual Data Virtualization across Disparate Data Sources
* Data Transformation e.g., from CSV, ODBC, JDBC Data Sources
* Federated Queries across various SPARQL Endpoints
* Small Data pattern for dynamically assembling data across a Semantic Web of Linked Data
